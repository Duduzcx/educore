# **App Name**: Progressive Access Portal

## Core Features:

- Login Screen: Secure login using email and password with 'remember me' option and password recovery.
- Social Registration: Progressive three-step registration: basic info, profile selection, and conditional fields.
- Profile Selection: Visual selection of user profile (ETEC Student, University Student, or Teacher).
- Conditional Data Collection: Collects additional data based on user profile, like school, course preferences, etc.
- AI-Powered Financial Aid Tool: Integrates a generative AI tool to determine potential fee exemptions based on user income data.
- Accessibility Integration: Integrates the VLibras button on the entry interface for accessibility.
- Smart Home: Carousel of 'Continue Studying' with the latest accessed trails.
- Learning Trails: Modular organization by subject, with cards of specific topics containing a logical sequence of lessons.
- Complete Classroom Environment: Integrated experience with video player, learning material (PDF), written summary, and mini-assessment.
- Live Streams: Real-time classes with integrated chat for questions and interaction with the teacher.
- Material Library: Central repository of handouts, exercise lists, and mind maps.
- 24/7 Pedagogical Support: Chat assistant for explaining concepts, step-by-step formula resolution, and immediate support at any time.
- Smart Notifications: Automatic alerts via Push and WhatsApp for critical events: Live class start, essay correction confirmation, and deadline reminders.

## Style Guidelines:

- Primary color: Deep Blue (#003049) to convey security and trust.
- Background color: Light Blue (#D4E0E9), desaturated and light to provide a clean, professional backdrop.
- Accent color: Gold (#D6AD60), slightly to the 'left' of blue on the color wheel, to highlight important interactive elements.
- Body and headline font: 'Inter', a sans-serif for modern legibility.
- Use clean, professional icons that represent security, education, and user profiles.
- Ensure a clean, responsive layout that adapts to different screen sizes, with clear information hierarchy.
- Subtle animations to guide users through the registration and login processes, enhancing user experience without being distracting.
