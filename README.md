## 🚀 Funcionalidades por Perfil

### 🎓 Aluno (Estudante)
- **Home Dinâmica**: Cards verticais premium com progresso real.
- **Sala de Aula Integrada**: Player de vídeo, materiais, quizzes e suporte Aurora IA.
- **Biblioteca Digital**: Acesso instantâneo a materiais de apoio diretamente na Home.
- **Simulador de Isenção**: Ferramenta social estratégica para vestibulares.

### 👨‍🏫 Professor (Mentor)
- **Studio Master**: Console de transmissão para lives (Meet/YouTube).
- **Gestão de Conteúdo**: Criação modular de trilhas e aulas.
- **Correção IA**: Sistema Aurora para análise de redações padrão ENEM.
- **Banco de Questões**: Repositório central com RPC de sorteio randômico.

### 🛡️ Gestor (Admin)
- **Dashboard Gestão 360**: Visão térmica de engajamento e performance da rede.
- **Workflow de Aprovação**: Controle de qualidade sobre todo conteúdo publicado.
- **Monitoramento de Risco**: Identificação precoce de alunos em risco de evasão.
- **Analytics BI**: Distribuição geográfica de matérias e interesses de carreira.

## 🎯 Configuração do Banco de Dados (Supabase)

Para o pleno funcionamento de todos os recursos, execute o arquivo `docs/database.sql` no seu SQL Editor. Ele configura:
1. Tabelas de relacionamento (Foreign Keys).
2. Políticas de segurança (RLS) modo demo.
3. Funções RPC para sorteio de questões.

## 🛠️ Arquitetura
- **Next.js 15**: Estabilidade de hidratação e performance.
- **Supabase**: Auth, PostgreSQL, Real-time e Storage.
- **Genkit**: Suporte pedagógico Aurora IA.
- **Shadcn UI + Tailwind**: Design industrial de alta fidelidade.
